## 使用

预留几个可能用到的设置

- 本主题无评论,可使用其他的评论系统如`artalk` `twikoo` `waline` `gitalk` 等
- 使用默认的CSS.在自定义CSS中可以使用官方的主题CSS
- 在文章页给有计数功能的评论系统预留HTML,实现文章浏览量,评论数等
- 在页脚增加HTML代码,可实现备案信息,统计代码等
